# 追加学習における既存知識への干渉を考慮した影響関数に基づくデータ合成 

##  概要 (Overview)
自動運転をはじめとするAIの実社会応用において、データ不足により学習が極めて困難なエッジケースが存在する「ロングテール問題」が最重要課題となっています。この対策として人工的な合成データを用いたデータ拡張が挙げられますが 、既存の重要知識に対する破滅的干渉を起こしてしまう問題がありました [。

本リポジトリは、**重要な既存知識に対して干渉を起こさないような合成データを生成する手法**の実装をまとめたものです 

##  提案手法 (Method)
あるクラスのデータを合成データで拡張したいが、他クラスの忘却を起こしたくないという問題設定に対し、生成モデルを誘導することで解決を図ります。具体的には、拡張したいクラス（クラスA）の正解率を改善し、守りたいクラス（クラスB）の正解率を悪化させない画像を生成します。

* **拡散モデル (DDPM) の採用**: 画像生成には Denoising Diffusion Probabilistic Model (DDPM) を用いています 
* **影響関数 (Influence Function) によるフィードバック**: データ追加による損失変化をヘシアンを用いて一次近似し、実際に再学習・検証せず、損失の変化を予測します 。
* **干渉を抑えた生成の誘導**: 生成モデルの逆拡散過程において、影響関数からのフィードバックに基づき、クラスAを改善する方向と、クラスBを悪化させない方向にサンプリング軌道を制御します。

## 実験と結果 (Experiments & Results)
CIFAR-10データセットを用い、特徴が類似し干渉が発生しやすい「鹿（クラスA: 拡張対象）」と「馬（クラスB: 保護対象）」で検証を行いました [cite: 248, 249]。

* **損失予測の妥当性確認**: 影響関数によるクラスBへの悪影響の予測値と実測値の相関係数は0.954であり、高精度で予測ができていることを確認しました [cite: 257, 413]。
* **保護対象クラスの悪化抑制**: 「馬クラスの悪化を抑える誘導 ($-g_B$)」によって生成した鹿画像を追加学習させたところ、無条件での生成や実データの追加時と比較して、馬クラスの正解率低下幅を最も抑制できました。
* **全体精度の向上**: 干渉を最小化して生成したデータは、保護したいクラスの悪化抑制に一定の効果がみられ、実データよりも全体精度が向上しました [cite: 296, 310]。

### 考察と今後の課題
* 鹿クラスの精度を向上させるように誘導しても、鹿クラスの正解率向上効果は低下する結果となりました。
* これは、誘導によって生成データの多様性が失われてしまったことが原因と考えられます。
* 今後の展望として、個々のデータを独立最適化するのではなく、追加するデータ全体を一度に集合最適化する手法を検討します。
